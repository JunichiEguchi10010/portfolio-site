# 画面画像仕様

## 制作・開発実績カード画像仕様

制作・開発実績セクション（`components/Works.tsx`）のカード画像を差し替える際の仕様です。

### ファイル配置と指定方法

| 項目 | 内容 |
| --- | --- |
| 画像配置先 | `public/images/works/` |
| 画像指定場所 | `data/works.ts` の `imageSrc` |
| 2枚目の画像パス（ファイル） | `public/images/works/inquiry-case-management-dashboard.png` |
| 2枚目の表示パス（`imageSrc` に指定する値） | `/images/works/inquiry-case-management-dashboard.png` |

`public/` 配下のファイルは、表示時には先頭の `public` を除いたパスで指定します。

### 推奨サイズ・アスペクト比

| 項目 | 内容 |
| --- | --- |
| 推奨画像サイズ | `920 × 690px` または `1200 × 900px` |
| アスペクト比 | `4:3` |

### 表示の実装

- `next/image` の `Image` コンポーネントを使用している
- `fill` 指定で親要素いっぱいに表示している
- 親要素は `aspect-[4/3]`（4:3 の枠を維持）
- `object-cover` のため、4:3 以外の画像は上下または左右が切れる可能性がある

### レスポンシブ・表示タイミング

- PC・スマホとも同じ画像を使用する（ブレークポイントで画像を切り替えない）
- スマホ時も画像は表示される
- 画像はアコーディオン展開時のみ表示される（カードを閉じた状態では非表示）

### 画像作成時の注意

- `object-cover` で端が切れることを前提に、重要な UI 要素や文字は中央寄りに配置する
