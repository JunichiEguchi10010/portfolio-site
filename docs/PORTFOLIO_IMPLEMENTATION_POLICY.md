# Portfolio Implementation Policy

## 目的

このリポジトリは、完成済みデザインをもとに、Next.js + TypeScript + Tailwind CSS でポートフォリオLPを実装するためのものです。

本サイトは、単なる自己紹介サイトではなく、AUTO_SCREENを中心とした「業務改善・データ活用・AI活用支援」の実績を伝えるポートフォリオとして作成します。

## 前提

* LPデザインは完成済み
* 新規デザイン提案は行わない
* 完成デザインの再現を最優先する
* AUTO_SCREEN本体をWebアプリ化するものではない
* まずは1ページ完結のLPとして実装する
* DB接続、認証、管理画面は初期実装では行わない

## 実装方針

* Next.js App Router を使用する
* TypeScript を使用する
* Tailwind CSS でスタイリングする
* セクション単位でReactコンポーネントに分割する
* まずPC表示を再現し、その後スマホ表示を調整する
* 不要なライブラリは追加しない
* アニメーションは最小限にする
* デザインの色、余白、文字サイズ、セクション順を勝手に変更しない

## ポートフォリオの見せ方

AUTO_SCREENは、投資助言ツールとしてではなく、以下のような判断支援システムとして紹介する。

* 大量データから候補を抽出する仕組み
* 指標・スコア・ランキングによる整理
* 日次運用できるデータ活用システム
* 人間の最終判断を補助するツール
* AIとCursorを活用して開発・改善した実例

## 重視する価値

* 業務課題を整理する力
* データを使って判断材料を作る力
* AIを実務に組み込む力
* 実装だけでなく運用・改善まで進める力
* 営業経験と業界理解を活かした現場目線

## 禁止事項

* 完成デザインと異なる独自デザインへの変更
* 不要なUIライブラリの導入
* バックエンド機能の追加
* 認証機能の追加
* AUTO_SCREENの実データや秘匿情報の掲載
* 投資成果を断定的に見せる表現
